### AIM:
Write a C program to find the ASCII value of a given character.

---

### ALGORITHM:
1. Take a character as input from the user  
2. Convert the character into its ASCII value using type casting  
3. Print the ASCII value using `printf()`  

---

### PROGRAM:

```c
#include<stdio.h>

int main()
{
    char ch;
    scanf("%c", &ch);
    printf("ASCII value of %c is %d", ch, ch);
}
```
